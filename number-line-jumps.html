Solution

We are dealing with a system of two linear equations: 

y1 = v1 * n + x1
y2 = v2 * n + x2

Where n is the number of kangaroo jumps and y1 and y2 are the kangaroo positions after n jumps. We want to find a solution to this system of equations when two kangaroos meet in the same position, so let’s compare the right sides of the equations: 

v1 * n + x1 = v2 * n + x2

After transformation: 

n = (x2 - x1) / (v1 - v2)

We cannot divide by zero, so we will not look for a solution this way if the speeds of the kangaroos are the same. In this case, the kangaroos will only meet if they start from the same position: x1 = x2. If n is the number of jumps, it must be a natural number. A negative number would mean that at their speeds they could have met in the past before they started from positions x1 and x2 but they will never meet after starting from their positions. A fractional number would mean that during the jump one kangaroo will overtake the other and they will never meet at the same position.

Let’s write this solution in the form of Java code: 

    public static String kangaroo(int x1, int v1, int x2, int v2) {
        if (v1 == v2) {
            return x1 == x2 ? "YES" : "NO";
        }
        double n = ((double) x2 - x1) / (v1 - v2);
        return n > 0 && n == (int) n ? "YES" : "NO";
    }
